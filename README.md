# NIMIZA 🌟

NIMIZA is an interactive edutainment platform designed to help children aged 3–8 develop emotional intelligence, curiosity, and essential life skills through stories, animated reels, and games.

## Features
- 🎬 Animated educational reels
- 📖 Interactive storybooks
- 🎮 Learning games
- 👦 Character-based learning (Nino, Miko, Zara)
- 🧠 Emotional intelligence development

- CSS

To make learning life skills fun through storytelling and interactive experiences.
